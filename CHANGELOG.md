# Changelog

All notable changes to this project will be documented in this file.

## [0.1.2] - 2026-04-06
### Added
- Column cursor and horizontal scrolling
- Vim-style navigation primitives
- Search support in shared state, UI, and plugin
- Cell/range/write address parsing for the native CLI
- CSV stdin loading and CSV write-back helpers

### Changed
- Improved native CLI help text and error messages
- Updated crate, README, and agent documentation to reflect current scope

## [0.1.1] - 2026-04-03
### Fixed
- Unified theme configuration between modules
- Fixed color rendering inconsistencies in UI
- Added proper error handling for UI rendering
- Added default configuration values
- Improved error messages and validation

## [0.1.0] - 2026-03-30
### Added
- Initial release of zellij-sheets
- Support for CSV and Excel file viewing
- Zellij plugin integration
- Native CLI for local testing
- Configurable display settings
